# FH6 휠스핀 자동화 v0.3.0 RC2

RC1에서 확인된 자동화 중복 실행 문제를 막는 긴급 수정 버전입니다.

## P0 수정

- 프로그램 자체를 동시에 두 번 실행하지 못하도록 차단
- 통합판·일반 단독판·슈퍼 단독판 사이에서 자동화 실행권을 하나만 허용
- 다른 자동화가 실행 중이면 F1 입력을 게임에 보내기 전에 차단하고 실행 정보를 표시
- 비정상 종료 뒤에는 운영체제가 실행 잠금을 자동 해제
- 프로젝트별 빌드 중간 폴더를 분리해 EXE가 다른 실험용 DLL을 참조할 가능성 제거

차량 판정과 희귀도 정책은 RC1과 동일합니다.

## 검증

- 별도 프로세스가 자동화 잠금을 보유한 상태에서 실행 차단 확인
- 기존 실행 종료 후 새 실행 허용 확인
- 정책 검사 23건 통과
- UI 배치 검사 통과
- Release 빌드 오류 0개

이 RC는 기존 스바루작 프로그램과 아직 결합되지 않은 독립 프로그램입니다.
