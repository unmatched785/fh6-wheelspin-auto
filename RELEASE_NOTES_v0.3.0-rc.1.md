# FH6 휠스핀 자동화 v0.3.0 RC1

일반 휠스핀과 슈퍼 휠스핀의 차량 판독·보호 정책을 확장한 공개 테스트 버전입니다.

## 주요 변경

- 일반·슈퍼 휠스핀에서 연식+제조사 조합으로 독점 차량 보호
- 일반 휠스핀 희귀도별 보관 옵션 추가
  - `노랑 차량`: 기본 활성화
  - `보라 차량`: 기본 비활성화
- 차량을 먼저 판독한 뒤 CR·경적·의류 등 비차량 상품을 구분하도록 순서 개선
- 중복차 팝업 연속 처리 안정화
- 한국어·영어 게임 UI 지원
- 기본 실행 모드는 자동이며, F4는 단계 실행에서만 사용

## 테스트 권장

처음에는 목표 1~3회로 실행하고 판정·입력이 맞는지 확인하세요. 예상과 다른 동작이 보이면 즉시 F2를 누르고 실행 파일 옆 `log` 폴더를 보존해 주세요.

이 RC는 기존 스바루작 프로그램과 아직 결합되지 않은 독립 프로그램입니다.
