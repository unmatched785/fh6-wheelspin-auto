# FH6 휠스핀 자동화 v0.1.0 RC1

일반 휠스핀과 슈퍼 휠스핀 자동화를 하나의 독립 프로그램으로 묶은 첫 공개 테스트 버전입니다.

## 포함된 기능

- 일반·슈퍼 휠스핀 선택 화면
- 기본 자동 실행
- 목표 횟수 기반 자연 종료
- 일반 휠스핀 중복차 자동 보관·판매
- 슈퍼 휠스핀 전체 배경 및 슬롯별 판정 정책
- 완료 후 메뉴 복귀
- F2 긴급정지, 로그 및 실패 화면 저장

## 처음 사용할 때

1. 게임에서 해당 휠스핀 카드에 포커스를 둡니다.
2. 목표를 1~3회로 설정합니다.
3. F1을 누르면 자동으로 시작합니다.

F4는 자동 실행에 필요하지 않습니다. `단계 실행`을 직접 선택했을 때만 다음 입력을 승인하는 점검용 키입니다.

## 알려진 범위

- 기존 스바루작 프로그램과 아직 결합하지 않았습니다.
- 일반 사용자는 슈퍼 휠스핀 정책을 `슬롯별 안전`으로 사용하세요.
- 독점 차량 연식·제조사 보호 기능은 아직 포함되지 않았습니다.

문제가 발생하면 F2로 정지하고 실행 파일 옆 `log` 폴더 전체를 제보해 주세요.
