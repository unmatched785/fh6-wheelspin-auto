# FH6 휠스핀 자동화

Forza Horizon 6의 **일반 휠스핀**과 **슈퍼 휠스핀**을 자동으로 처리하는 독립 프로그램입니다.

현재 버전은 실제 사용자 환경을 넓혀 검증하기 위한 **공개 RC(출시 후보)** 입니다. 기존 스바루작 프로그램과는 결합되지 않았습니다.

## 주요 기능

- 일반 휠스핀과 슈퍼 휠스핀을 하나의 프로그램에서 선택
- 목표 횟수만큼 자동 실행하고 실제 결과 확정 횟수로 카운트
- 빠른 정지, 결과색 판정, 중복차 팝업 처리 자동화
- 일반 휠스핀 완료 후 오픈월드 복귀
- 슈퍼 휠스핀 슬롯별 색상 판정
- F2 긴급정지와 실행 로그·화면 캡처 저장

## 사용법

1. [Releases](../../releases)에서 최신 ZIP을 내려받아 압축을 풉니다.
2. `fh6-wheelspin-auto.exe`를 실행합니다.
3. 일반 휠스핀 또는 슈퍼 휠스핀을 선택합니다.
4. 게임의 `나의 HORIZON` 화면에서 사용할 휠스핀 카드에 포커스를 둡니다.
5. 목표 횟수를 설정하고 **F1**을 누릅니다.

기본값은 **자동 실행**입니다.

### 단축키

- `F1`: 자동 실행 시작
- `F2`: 긴급정지
- `F3`: 현재 실행 화면 종료
- `F4`: 단계 실행에서만 다음 입력 승인

`F4`는 자동 실행에 필요하지 않습니다. 화면 판정과 다음 입력을 한 단계씩 확인하려고 사용자가 직접 `단계 실행`을 선택했을 때만 사용합니다.

## 슈퍼 휠스핀 정책

- 일반 사용자는 `슬롯별 안전 (일반 사용자)`를 권장합니다.
- 모든 차량·경적·의류를 이미 보유한 계정만 `슬롯순 정확 (모든 보유품 완성)`을 사용하세요.
- 처음에는 목표 1~3회로 정상 동작을 확인한 뒤 횟수를 늘리세요.

## RC 주의사항

- FHD와 4K 환경을 중심으로 개발·검증했습니다.
- 예상과 다른 화면이나 입력이 보이면 즉시 `F2`를 누르세요.
- 문제가 생기면 실행 파일 옆 `log` 폴더 전체와 해상도·FPS·창 모드를 함께 알려주세요.
- 독점 차량을 연식·제조사로 보호하는 기능은 아직 포함되지 않았습니다.

## Windows 경고

코드 서명이 적용되지 않은 개인 배포 프로그램이라 Windows SmartScreen 경고가 나타날 수 있습니다. 반드시 이 저장소의 Releases에 첨부된 파일과 SHA-256을 확인하세요.
